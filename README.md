# TextUtils 🛠️

**TextUtils** is a simple web-based text utility tool built using **Django**. It provides multiple text processing features to simplify and format text.

---

## 🚀 Features

- ✅ Remove Punctuation  
- ✅ Convert Text to UPPERCASE  
- ✅ Remove Extra Spaces  
- ✅ Count Characters  
- ✅ New Line Remover  
- ✅ Analyze Your Text in One Click
